LUNIVA – AI POWERED HEALTHCARE ASSISTANT

Luniva is an AI-powered healthcare assistance platform designed to help users access essential health resources quickly and efficiently. The platform focuses on improving healthcare accessibility by providing tools such as symptom checking, emergency support, health education, and health tracking in a simple and user-friendly interface.

This project was developed as part of a hackathon to explore how artificial intelligence and modern web technologies can improve healthcare support and accessibility.

FEATURES

AI Health Assistant
An interactive AI assistant that helps users get basic healthcare information and guidance.

Symptom Checker
Allows users to track and analyze symptoms to better understand possible health conditions.

Emergency Assistance
Provides quick access to emergency resources and important healthcare information.

Health Education
Educational resources that help users understand diseases, symptoms, and preventive healthcare practices.

Health Tracking
Users can monitor health activities and maintain a record of their symptoms and wellness data.

Privacy Focused
User privacy and security are considered during the design of the platform.

INSTALLATION AND SETUP

Follow the steps below to run the project locally.

1.Clone the repository

git clone https://github.com/misananya07/lunivaa.git

2.Navigate to the project directory

cd luniva

3.Install dependencies

npm install

This will install all required dependencies and create the node_modules folder.

4.Start the development server

npm run dev

5.Open the application in your browser

http://localhost:3000
